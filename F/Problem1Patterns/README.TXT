A pattern catalog consists of the following design patterns:

1.      Strategy

2.      Composite

3.      Adapter

4.      Publisher-Subscriber

5.      Abstract Factory

6.      Master-Slave

7.      Wolf-Sheep

8.      Proxy

Match each of the following scenarios with the pattern from this catalog that is best suited to model it.

The answer is 4. Publisher-Subscriber   a.      A controller needs to react to button clicks.

The answer is 2.Composite   b.      A programming language provides a collection of simple commands such as add and multiply, but also provides a block command that allows programmers to group multiple commands into a single, sequentially executed command. For example: {add; mul; { add; add }}

The answer is 6.Master Slave   c.  An executive needs a secretary to screen calls. The secretary needs and secretary to pre-screen calls. The secretary's secretary needs a secretary to pre-pre-screen calls.

The answer is 5. Abstract Factory   d.      A word processor provides a template for assembling textbooks-- title page, table of contents, chapters, references, index. But not the actual content.

The answer is 1. Strategy    e.      A military drone needs the ability to quickly change its mission from attack to rescue to observe etc
